Brands

Description A list of active Happy Melly brands
Request GET /api/v2/brands
Request path parameters none
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of brand objects, with properties:

  • unique_name
  • name
  • image - URL of a brand's logo, excluding protocol/host
  • tagline
  • products - a number of products belongs to the brand
Errors (HTTP status) none
Example request GET /api/v2/brands?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

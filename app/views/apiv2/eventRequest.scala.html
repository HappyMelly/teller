Event request creation

Description Adds new event request to the system
Request POST /api/v2/eventrequest/brand/:code
Request path parameters brand a brand code an adding event request belongs to, e.g. mgt30
Request POST parameters (not member)
  • country - ISO 8166-2 country code, string
  • city - optional, list of cities divided by commas, ex. Orlando, London
  • language - list of languages divided by commas, ex. Russian, English
  • start_date - optional, string, YYYY-mm-DD
  • end_date - optional, string, YYYY-mm-DD
  • comment - optional, string
  • name - name of the person who makes this request, string
  • email - email of the person who makes this request, string
  • number_of_participants - number of participants, int
Response content type application/json; charset=utf-8
Response body

A JSON object with properties:

  • request_id - unique identifier for a created request
Errors (HTTP status)

400 Bad Request if parameters are wrong, with a JSON object, containing errors for all parameters

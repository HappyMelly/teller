Events

The events resource is a list of events for a particular brand, optionally filtered by several parameters.

Request GET /api/v2/brand/{brand}/events
Request path parameters brand a brand code to return events for, e.g. mgt30
Request query string parameters (optional)
  • future - filter events by time: true for future events and false for past events
  • public - filter events by publicity: true for public events and false for private events
  • archived - filter events by 'Archived' flag
  • facilitatorId - filter events by facilitator
  • countryCode - filter events by ISO 8166-2 country code, e.g. AT
  • eventType - filter events by an event type
Response content type application/json; charset=utf-8
Response body

A JSON array of events objects, with properties:

  • id - an unique identifier for an event
  • title
  • type - see Event Types
  • typeName - name of event type
  • description
  • specialAttention
  • spokenLanguages - an array of languages
  • materialsLanguage
  • start
  • end
  • totalHours
  • hoursPerDay
  • facilitators - an array of objects
    • id - an unique identifier for a person
    • href - URL of the person details resource, excluding protocol/host and starting with /api
    • first_name
    • last_name
    • photo
    • city
    • country - ISO 8166-2 country code
  • city
  • country - ISO 8166-2 country code
  • website
  • registrationPage
  • rating - event's rating calculated from its evaluations
  • confirmed - true if the event is confirmed
  • free - true if the event is free
Errors (HTTP status)

404 Not Found if the given brand code is not defined
Examples request GET /api/v2/brand/mgt30/events?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2
GET /api/v2/brand/mgt30/events?future=true&public=true&api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

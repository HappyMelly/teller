Brand fees

Description A list of event license fees for the given brand
Request GET /api/v2/fees/:brand
Request path parameters brand - brand code, ex. MGT30
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of fee objects, with properties:

  • id
  • brand
  • country - ISO 8166-2 country code
  • fee - fee currency and amount, ex. EUR 100.00
Errors (HTTP status) none
Example GET /api/v2/fees/mgt30

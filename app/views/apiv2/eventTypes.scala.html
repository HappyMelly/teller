Event Types

The event types resource is a list of event types for a particular brand.

Request GET /api/v2/brand/{brand}/types
Request path parameters brand a brand code to return event types for, e.g. mgt30
Response content type application/json; charset=utf-8
Response body

A JSON array of event types objects, with properties:

  • id - unique identification number of the type
  • brand - a brand code, e.g. MGT30
  • name
  • defaultTitle
Errors (HTTP status)

404 Not Found if the given brand code is not defined

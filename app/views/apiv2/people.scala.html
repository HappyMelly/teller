People

Description A list of Happy Melly People, optionally filtered by their activity status and names
Request GET /api/v2/people
Request path parameters none
Request query string parameters query string to filter people by names
Response content type application/json; charset=utf-8
Response body

A JSON array of person objects, with properties:

  • id - a numeric unique identifier of a person, e.g. 42
  • unique_name - an unique identifier of a person, e.g. sergey.kotlov
  • name
  • photo
  • city
  • country - ISO 8166-2 country code
Errors (HTTP status) none

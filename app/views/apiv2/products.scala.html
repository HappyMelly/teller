@()

Products

Description A list of Happy Melly products
Request GET /api/v2/products
Request path parameters none
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of product objects, with properties:

  • href - URL of the product details resource, excluding protocol/host and starting with /api
  • title
  • subtitle
  • image - URL of a product's image, excluding protocol/host
  • brands - array of objects with properties:
    • code
    • name
  • category
Errors (HTTP status) none
Example request GET /api/v2/products?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

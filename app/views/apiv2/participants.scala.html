Participants

Description A list of participants for a particular event
Request POST /api/v2/event/{event}/participants
Request path parameters event an event id to return participants for, e.g. 40
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of person objects, with properties:

  • id - an unique identified for a person
  • unique_name
  • first_name
  • last_name
  • photo
  • country - ISO 8166-2 country code
Errors (HTTP status)

400 Bad Request if parameters are wrong, with a JSON object, containing errors for all parameters
Example request GET /api/v2/event/40/participants?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

Experiments

List of all experiments run by Happy Melly members

Request GET /api/v2/experiments
Request path parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of experiment objects, with properties:

  • memberId - id of a member who runs this experiment
  • name
  • description
  • url - optional link
  • image - optional image
Errors (HTTP status) none
Example request GET /api/v2/experiments?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

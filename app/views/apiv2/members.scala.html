Members

Description A list of Happy Melly members
Request GET /api/v2/members
Request path parameters none
Request query string parameters funder optional parameter. If true, only funders are returned. If false, only supporters. If empty, all members.
Response content type application/json; charset=utf-8
Response body

A JSON array of member objects, with properties:

  • id
  • name
  • type - "person" or "org"
  • funder
  • image - url with avatar or logo
Errors (HTTP status) none
Example request GET /api/v2/members?api_token=4d5d132664a3f42e9a04a53ca701f36a5fba53a2

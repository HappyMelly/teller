Brand details

The brand details resource provides more details about an entry in the list.

Request GET /api/v2/brand/{brandCode}
GET /api/v2/brand/{brandUniqueName}
Request path parameters brandCode unique identifier for the brand, e.g. DARE
brandUniqueName unique identifier for the brand, e.g. management30
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON object, with brand properties:

  • code
  • unique_name
  • name
  • tagline
  • image - URL of a brand's logo, excluding protocol/host
  • description
  • coordinator - a JSON object with person's properties:
    • href - URL of the person details resource, excluding protocol/host and starting with /api
    • name
    • photo
    • city
    • country - ISO 8166-2 country code
  • contact_info - a JSON object with properties:
    • email
    • skype
    • phone
  • social_profile - a JSON object with properties:
    • facebook
    • twitter
    • google_plus
    • linkedin
  • website
  • blog
  • products - a list of JSON objects
    • href - URL of the product details resource, excluding protocol/host and starting with /api
    • title
    • image - URL of a product's image, excluding protocol/host
    • brands - array of objects with properties:
      • code
      • name
    • category
Errors (HTTP status)

404 Not Found if the specified brand is not found
Examples GET /api/v2/brand/mgt30

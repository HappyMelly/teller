Event

The event details resource provides more details about an entry in the list.

Request GET /api/v2/event/{eventId}
Request path parameters

eventId unique identifier for the event, e.g. 42
Request query string parameters (optional) none
Response content type application/json; charset=utf-8
Response body

A JSON array of events objects, with properties:

  • brand - a code for a brand
  • type - see Event Types
  • title
  • description
  • spokenLanguages - an array of languages
  • materialsLanguage
  • start
  • end
  • hoursPerDay
  • totalHours
  • facilitators - an array of objects
    • href - URL of the person details resource, excluding protocol/host and starting with /api
    • name
    • photo
    • city
    • country - ISO 8166-2 country code
  • city
  • country - ISO 8166-2 country code
  • website
  • registrationPage
  • public - true if the event is public
  • archived - true if the event is archived
Errors (HTTP status)

404 Not Found if the given brand code is not defined
Examples GET /api/v2/event/1

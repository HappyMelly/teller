Certificate generation

Description Generate new certificate for a participant
Request POST /api/v1/certificate/:evaluationId
Request path parameters

evaluationId unique identifier for the evaluation, e.g. 5
Request POST parameters none
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON object with properties:

  • success - Certificate generated
Errors (HTTP status)

404 Not Found if parameters are wrong or the evaluation doesn't exist

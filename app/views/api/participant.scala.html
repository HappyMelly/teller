Participant creation

Description Add new participant to the system
Request POST /api/v1/participant
Request POST parameters
  • eventId unique identifier for the event, numeric
  • emailAddress string
  • lastName string
  • firstName string
  • country - ISO 8166-2 country code, string
  • city string
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON object with properties:

  • participantId - unique identifier for a created participant
Errors (HTTP status)

400 Bad Request if parameters are wrong, with a JSON object, containing errors for all parameters

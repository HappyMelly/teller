Facilitators

Description A list of facilitators for a particular brand
Request GET /api/v1/brand/{brand}/facilitators
Request path parameters brand a brand code to return events for, e.g. mgt30
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of person objects, with properties:

  • id - an unique identified for a person
  • first_name
  • last_name
  • photo
  • city
  • country - ISO 8166-2 country code
  • languages - a list of languages the facilitator speaks
  • countries - a list of ISO 8166-2 country codes where the facilitator works in
Errors (HTTP status) none
Examples GET /api/v1/brand/mgt30/facilitators

Events By Country

A list with a number of events per country.

Request GET /api/v1/events/{brand}/grouped/country
Request path parameters brand a brand code to return events for, e.g. mgt30
Request query string parameters (optional) none
Response content type application/json; charset=utf-8
Response body

A JSON array of objects, with properties:

  • country - ISO 8166-2 country code
  • eventsNumber
Errors (HTTP status)

404 Not Found if the given brand code is not defined
Examples GET /api/v1/events/mgt30/grouped/country

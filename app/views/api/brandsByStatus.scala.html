Brands By Status

Description A list of Happy Melly brands, grouped by statuses
Request GET /api/v1/brands/grouped/status
Request path parameters none
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON array of brand objects, with properties:

  • status
  • brands - a list of objects
    • href - URL of the brand details resource, excluding protocol/host and starting with /api
    • name
    • image - URL of a brand's logo, excluding protocol/host
    • description
    • status
Errors (HTTP status) none
Example GET /api/v1/brands/grouped/status

Member details

The member details resource provides more details about an entry in the list.

Request GET /api/v1/member/{memberId}
Request path parameters memberId unique identifier for the member, e.g. 42
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON object, with member properties:

  • id
  • type - "person" or "org"
  • funder
  • org - if a member is an organization, this attribute contains Organization details
  • person - if a member is a person, this attribute contains Person details
Errors (HTTP status)

404 Not Found if the specified member is not found
Examples GET /api/v1/member/1

Product details

The product details resource provides more details about an entry in the list.

Request GET /api/v1/product/{productId}
Request path parameters productId unique identifier for the product, e.g. 42
Request query string parameters none
Response content type application/json; charset=utf-8
Response body

A JSON object, with product properties:

  • title
  • subtitle
  • url
  • description
  • cta_url - URL for a button 'Call To Action'
  • cta_text - text for a button 'Call To Action'
  • image - URL of a product's image, excluding protocol/host
  • brands - array of objects with properties:
    • code
    • name
  • category
  • parent - URL of the parent product details resource, excluding protocol/host and starting with /api
  • contributors - array of objects with properties:
    • name - a name of a contributor
    • unique_name - an unique name of a contributor
    • href - URL of the person details resource, excluding protocol/host and starting with /api
    • photo
    • role
Errors (HTTP status)

404 Not Found if the specified product is not found
Examples GET /api/v1/product/1

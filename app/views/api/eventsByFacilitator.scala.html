Events By Facilitator

The events resource is a list of events for a particular brand and a parcticular facilitator, optionally filtered by several parameters.

Request GET /api/v1/events/{brand}/facilitator/{facilitatorId}
Request path parameters
  • brand a brand code to return events for, e.g. mgt30
  • facilitatorId an unique identifier of a facilitator to return events for
Request query string parameters (optional)
  • future - filter events by time: true for future events and false for past events
  • public - filter events by publicity: true for public events and false for private events
Response content type application/json; charset=utf-8
Response body

A JSON array of events objects, with properties:

  • title
  • description
  • spokenLanguage
  • start
  • end
  • totalHours
  • city
  • country - ISO 8166-2 country code
  • website
  • registrationPage
Errors (HTTP status)

404 Not Found if the given brand code is not defined
Examples GET /api/v1/events/mgt30/facilitator/1
GET /api/v1/events/mgt30/facilitator/1?future=true&public=true
